# @rudeigerc/shiki-transformer-title

## Installation

```shell
npm install @rudeigerc/shiki-transformer-title
```

## Usage

```css
.shiki-code-block-title {
  margin-top: 1rem;
  padding-left: 1rem;
  padding-right: 1rem;
  padding-top: 0.5rem;
  padding-bottom: 0.5rem;
  font-size: 0.875rem;
  line-height: 1.25rem;
  font-weight: 500;
}
```

With Tailwind CSS:

```css
.shiki-code-block-title {
  @apply mt-4 px-4 py-2 text-sm font-medium;
}
```
